Strong Password Checking using Hash Tables 
<br> 
<br>
This project is implementing an application that has a strong password checker by utilizing Hash Tables to check if a password satisfies the password requirements quickly. The program utilizes a 10,000 word list of common words and uses hash functions for finding common weak passwords. In addition, the program will compare two hash function approaches and two methods for resolving collisions in order to find the fastest approach.
<br>

## Features 
- Password validation according to certain rules for security 
- Finds dictionary-based weak passwords 
- Uses two methods to implement hash tables: 
- Separate chaining 
- Linear probing 
- Compare two hash functions: 
 - The early version of java's hashCode() method which has a multiplier of 37 and uses a "skipping" technique 
 - The modern version of java's hashCode() method which has a multiplier of 31 
- Determines the cost of searching in the hash table (number of comparisons)
